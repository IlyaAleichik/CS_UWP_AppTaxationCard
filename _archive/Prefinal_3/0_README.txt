ВНИМАНИЕ!!!
Для установки данного программного средства версия Windows 10 должна быть не ниже 1809 (17763.xxx)

Инструкция по установке  AppTaxationCard v1.0.1.0
Все нижеперечисленные файлы вы найдете в коревом каталоге папки "WapProjTemplate_1.0.1.0_Package" и в папке "Dependencies"

1.Установите Microsoft.NET.CoreFramework.Debug.2.2 

2.Установите Microsoft.NET.CoreRuntime.2.2.

3 Установите сертификат WapProjTemplate_1.0.1.0_x64 ввыполнив следующие шаги 

- откройте файл двойным щелчком мыши по WapProjTemplate_1.0.1.0_x64.cer
- нажмите "Установить сертификат..." откроется Мастер импорта сертификатов
- выберите Локальный компьютер и нажмите далее
- нажмите на Поместить все сертификаты в следющее хранилище и нажмите Обзор
- выберите каталог Доверенные корневые центры сертификации и нажмите ОК.
- нажмите Далее и Готово, появится окошко с сообщением "Импорт произведен успешно" нажимте ОК.
